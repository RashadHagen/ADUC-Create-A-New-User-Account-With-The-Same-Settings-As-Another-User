<h1>Active Directory Users and Computers (ADUC) - Create A New User Account With The Same Settings As Another User</h1>


<h2 style="font-family: Arial, sans-serif; font-size: 20px; font-weight: bold; margin-top: 24px; margin-bottom: 12px;">
⏹️ Description</h2>

<p style="font-family: Georgia, serif; font-size: 16px; margin-top: 12px; margin-bottom: 12px;">
This project goes over how to create a new user account with the same settings as another user in Active Directory Users and Computers. When an administrator creates a new user account with the same settings as another user in Active Directory Users and Computers, it allows for the rapid deployment of similar user profiles and access levels across a domain.
</b>
